# kokoro-epub

> **Disclaimer:** Only use material from the public domain.

Convert EPUB, TXT, or PDF files to audio using Python and PyQt.

## Quick Start

```bash
# Setup
python3.11 -m venv kokoro-env
source kokoro-env/bin/activate
pip install -r requirements.txt

# Run
cd app
python pyqt_app.py
```
